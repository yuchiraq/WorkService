# Проект на Go с Gin

Это простой веб-сервер, созданный с использованием фреймворка Gin.

## Структура проекта

- `cmd/server/main.go`: Основная точка входа в приложение.
- `internal/`: Внутренняя логика приложения.
  - `api/`: Обработчики HTTP-запросов.
  - `models/`: Структуры данных (модели).
  - `router/`: Настройка маршрутов.
  - `storage/`: Функции для работы с хранилищем данных.
- `data/`: JSON-файлы, используемые в качестве хранилища.
- `web/`: Файлы для фронтенда.
  - `templates/`: HTML-шаблоны.

## Запуск проекта

1.  **Установите зависимости:**

    ```bash
    go mod tidy
    ```

2.  **Запустите сервер:**

    ```bash
    go run ./cmd/server
    ```

    Сервер будет доступен по адресу `http://localhost:8099`.

## Доступные эндпоинты

### API (группа `/api`)

*   **GET /api/ping**: Проверка доступности сервера. Возвращает `{"message":"pong"}`.
*   **POST /api/users**: Создание нового пользователя. Тело запроса:

    ```json
    {
        "username": "testuser",
        "password": "testpass"
    }
    ```

*   **POST /api/articles**: Создание новой статьи. Тело запроса:

    ```json
    {
        "title": "Моя первая статья",
        "content": "Это содержимое моей первой статьи."
    }
    ```

*   **GET /api/articles/:id**: Получение статьи по ее ID.
*   **PUT /api/articles/:id**: Обновление статьи по ее ID. Тело запроса аналогично созданию статьи.

### Web (группа `/web`)

*   **GET /web/login**: Страница входа в систему.
*   **POST /web/login**: Обработка данных формы входа.
*   **GET /web/dashboard**: Панель управления (требует аутентификации, которая пока не реализована).
