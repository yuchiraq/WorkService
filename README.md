# Веб-приложение на Go и Gin

Это простое веб-приложение, созданное с использованием языка программирования Go и фреймворка Gin. Приложение предоставляет базовый REST API для управления пользователями и статьями, а также веб-интерфейс для взаимодействия с этими данными.

## Структура проекта

- **/cmd/server**: Основной пакет приложения, который запускает веб-сервер.
- **/internal/api**: Обработчики HTTP-запросов (контроллеры).
- **/internal/models**: Структуры данных (модели) для пользователей и статей.
- **/internal/router**: Настройка маршрутизации (роутер) для веб-приложения.
- **/internal/storage**: Пакет для работы с хранилищем данных. В текущей версии данные хранятся в JSON-файлах.
- **/data**: Директория для хранения JSON-файлов (`users.json`, `articles.json`).
- **/web/templates**: HTML-шаблоны для веб-интерфейса.
- **/web/static**: Статические файлы (CSS, JavaScript, изображения).

## Функциональность

- **Пользователи**:
    - `POST /api/users`: Создание нового пользователя.
- **Статьи**:
    - `POST /api/articles`: Создание новой статьи.
    - `GET /api/articles/:id`: Получение статьи по ID.
    - `PUT /api/articles/:id`: Обновление статьи по ID.

## Запуск приложения

1.  **Клонируйте репозиторий** (если вы еще этого не сделали):
    ```bash
    git clone <URL-адрес репозитория>
    cd <название-директории>
    ```

2.  **Установите зависимости**:
    ```bash
    go mod tidy
    ```

3.  **Запустите сервер**:
    ```bash
    go run ./cmd/server
    ```
    Сервер будет запущен на порту `8099`.
